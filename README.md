# SmartLend

SmartLend is a full-stack loan management system built with ASP.NET Core, React, and FastAPI. The application enables customers to submit loan requests while administrators review and approve or reject applications. A Python-based machine learning service predicts the applicant's credit risk.

---

## Features

- JWT Authentication
- Role-based Authorization (Customer / Admin)
- Loan Application Submission
- Risk Score Prediction using Machine Learning
- Admin Dashboard
- Customer Dashboard
- Responsive Material UI Interface
- RESTful API
- SQL Server Database
- Docker Support
